Welcome to CatCraft!

Before opening the source code, please install TurboWarp and use it to open the project. Do NOT use Scratch!

This is a 2D mini-game made with TurboWarp, but I believe it will become more popular in the near future (:

Use W, A, D to jump, move left, and move right respectively. Gently slide the mouse to move the pickaxe. Aim the pickaxe at a block to mine it!
